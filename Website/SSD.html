What is an SSD?

SSD stands for Solid State Drive and it is used for storing data in a much better way than what we have been doing so far in our devices. An SSD is different to how an Hard Disk Drive (HDD) functions. It is much like those small USB memory stick which uses microchips to store data and it does not have any READ/WRITE head like HDD. This difference makes SSD much faster and a preferable product to use for our devices.
